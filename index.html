HTML

  • The "head" element contains information about the webpage.
  • The "body" element represents the visible content shown to the user.
  • The "style" element defines the CSS styles associated with a website.
  • The "base" element defines the base URL for a webpage.
  • The "link" element connects an external resource to the HTML document.
  • The "meta" element defines metadata such as the character set, description, keywords, author, and viewport.
  • The "script" element embeds client-side scripts in the HTML document.
  • h1, h2, h3, h4, h5, h6 represent the level of heading a given text block represents
  • the "p" element represents a paragraph or block of text.
  • "ul", "ol", "li" elements represent unordered lists, ordered lists, and list items.
  • the "img" element contains info about images that are displayed.
  • "img" elements need "src" attribute bc it defines the location of the image.
  • "br" creates a line of empty space.
  • "a" element creates links to webpages. "href" points to the URL for the link.

Git

  • git status: checks what branch we are currently on
  • git checkout -b *branch-name*: creates a new branch and switches to it
  • git branch: list your branches
  • git init: init an existing directory as a Git repo
  • git clone URL: retrieve repo from URL
  • git add -A: commit all files to working branch
  • git commit -m "add text here": commit changes with message

JavaScript

  • a variable is a container that allows us to store data in our code (ie- "function", "const", "let")
  • "const" keyword allows us to create variables that cannot be reassigned a value.
  • "=" to assign specific data that we want stored in our variable.
  • variables contain "strings" that are contained w quotation marks.
  • each line of javascript code ends in a semicolon.
  • "if" statements are a way of introducing decision-making into code.
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal.
  • An array is a single variable that is used to hold a group of data.
  • Arrays are useful tools for storing multiple items with different data types in a single variable, but not all arrays must have multiple data types.
  • A "for" loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A "for" loop contains the following three important statements: the starting point for our loop ("let x = 0"), the condition ("x < shapes.length"), and what allows the array to iterate over each item ("x+1 or x++")
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
