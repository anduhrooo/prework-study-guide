HTML

  • The "head" element contains information about the webpage.
  • The "body" element represents the visible content shown to the user.
  • The "style" element defines the CSS styles associated with a website.
  • The "base" element defines the base URL for a webpage.
  • The "link" element connects an external resource to the HTML document.
  • The "meta" element defines metadata such as the character set, description, keywords, author, and viewport.
  • The "script" element embeds client-side scripts in the HTML document.
  • h1, h2, h3, h4, h5, h6 represent the level of heading a given text block represents
  • the "p" element represents a paragraph or block of text.
  • "ul", "ol", "li" elements represent unordered lists, ordered lists, and list items.
  • the "img" element contains info about images that are displayed.
  • "img" elements need "src" attribute bc it defines the location of the image.
  • "br" creates a line of empty space.
  • "a" element creates links to webpages. "href" points to the URL for the link.

Git

  • git status: checks what branch we are currently on
  • git checkout -b *branch-name*: creates a new branch and switches to it
  • git branch: list your branches
  • git init: init an existing directory as a Git repo
  • git clone URL: retrieve repo from URL
  • git add -A: commit all files to working branch
  • git commit -m "add text here": commit changes with message

    • Saving Work to Github

  • navigate to "prework-study-guide" directory.
  • type "git status" to check which branch you're in.
  • "git checkout main" to switch to main branch.
  • pull down latest main branch by entering "git pull origin main".
  • create new branch by typing "git checkout -b *name of new branch*".
  • "git status" to check branch.
  • "code ." to open and work in VS CODE.
  • "git add -A" to stage all changes to working branch.
  • "git commit -m "add message here"" to commit changes to repo.
  • "git pull origin main" to pull in the base branch.
  • "git push origin *branch-name*" to push up all changes to repo.
